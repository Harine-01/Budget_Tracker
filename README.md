# Budget_Tracker
A simple budget tracker web app to manage income and expenses using HTML, CSS, JavaScript, and localStorage.

## Description
A simple budget tracker web application that helps users manage their personal finances by recording income and expenses. The application calculates total income, total expenses, and balance, and stores data using localStorage so it remains available after page reload.

## Features
- Add income and expense transactions
- Display a list of all transactions
- Delete transactions
- Automatic calculation of total income, total expenses, and balance
- Persistent data storage using localStorage
- Color-coded transactions (green for income, red for expense)
- Filter transactions (All / Income / Expense)
- Simple chart visualization of income vs expense

## Technologies Used
- HTML
- CSS
- JavaScript
- Chart.js (for visualization)
